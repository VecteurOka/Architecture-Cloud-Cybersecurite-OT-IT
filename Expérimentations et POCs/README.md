
**Ce dossier regroupe mes expérimentations techniques et preuves de concept (POCs)** réalisées pour tester, valider ou explorer des solutions innovantes dans les domaines suivants :
- **Intégration OT/IT** : Protocoles (OPC UA, MQTT), passerelles, interopérabilité.
- **Cloud Industriel** : Déploiement d'applications IIoT, Edge Computing, architectures hybrides.
- **Cybersécurité** : Tests de vulnérabilités, détection d'intrusions, durcissement de systèmes.
- **Industrie 4.0** : Digital Twins, analyse de données temps réel, automatisation intelligente.

Chaque POC est documenté pour permettre sa **répétabilité, son analyse et son amélioration**.

# Architecture OT/IT, Cloud, Cybersécurité et Industrie 4.0

**Dépôt centralisant mes articles techniques, projets académiques, expérimentations (POCs) et ressources de veille sur l'Industrie 4.0, l'architecture OT/IT, le Cloud et la cybersécurité industrielle.**

---

## 📌 À propos de ce dépôt
Ce dépôt a pour objectif de :
- **Partager** mes articles techniques et analyses sur les technologies i4.0.
- **Archiver** mes projets académiques et expérimentations (POCs).
- **Documenter** mes recherches et ressources de veille sur l'OT/IT, le Cloud et la cybersécurité.
- **Faciliter** la réutilisation et la collaboration sur ces sujets.

---

## 📂 Structure du dépôt
- `00_Documentation/` : Guides et normes
- `01_Projets-Academiques/` : Projets universitaires
- `02_Articles-et-Recherches/` : Publications et recherches
- `03_Expérimentations et POCs/` : Lab
- `04_Veille I4.0/` : Ressources externes

---

## 🛠 Technologies Utilisées
- Outils OT/IT : Siemens TIA Portal, Ignition SCADA
- Cloud : AWS, Azure, Google Cloud
- Cybersécurité : Nessus, Wireshark, ISO 27001
- Langages : Python, Java, html
